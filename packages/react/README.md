# @perfectest/react

React components and hooks.

## Installation

```bash
npm install @perfectest/react
# or
pnpm add @perfectest/react
```

## Usage

```tsx
import { Button } from "@perfectest/react";
// or import specific components
import { Button } from "@perfectest/react/button";
```

## Peer Dependencies

- react >= 16.0.0
- react-dom >= 16.0.0
